Le chat ouvrit les yeux 
Le soleil y entra.
Le chat ferma les yeux
Le soleil y resta.

Voilà pourquoi le soir,
Quand le chat se réveille,
J'aperçois dans le noir
Deux morceaux de soleil.

Maurice Carême

Web scraping pour les nuls (appliqué à Bing Desktop )

J’aime Bing Desktop que j’utilise essentiellement … comme fond d’écran. Les photos sont magnifiques et se renouvellent tous les jours. Les découvrir chaque matin est une invitation au voyage.

Il m’était très fréquent de vouloir en savoir plus sur la photo du jour : quelle est cette fleur, cette animal, ce lieu ?

Jusqu’alors, un requête Google “Bing picture of the day” m’orientait vers la réponse, mais pas toujours de manière immédiate.

Voilà pourquoi j’ai créé cette page avec le tableur de google. Elle récupère ici et là quelques informations sur la photo et les affiche dans une page épurée, accompagnée parfois de quelques commentaires glanés sur le Web . C’est tout.

Pour les curieux, voici le source (pensez à faire fichier -> faire une copie pour visualiser les formules; c’est fou ce que l’on peut faire avec le langage de scripting des googles apps).
